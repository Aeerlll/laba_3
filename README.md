# Анализ температуры за месяц

Windows Forms-приложение для анализа температурных данных.

## Возможности

- загрузка CSV файла через окно выбора
- вывод данных в таблице
- построение графика по дням
- минимальная температура — синим
- максимальная температура — красным
- расчет дня с самым сильным и самым слабым перепадом
- прогноз на N дней методом скользящего среднего
- экспорт графика в PNG
- выбор периода по датам
- масштабирование графика выделением области мышью

## Запуск

1. Открыть проект в Visual Studio
2. Убедиться, что выбран тип проекта Windows Forms App (.NET Framework)
3. Собрать проект
4. Запустить `Program.cs`

## Формат CSV

Колонки:

- date
- min_temp
- max_temp
- avg_temp
- weather

Дата должна быть в формате `yyyy-MM-dd`.
