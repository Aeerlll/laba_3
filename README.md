# Анализ температуры за месяц

Windows Forms-приложение для анализа температурных данных.

## Возможности

- загрузка CSV файла через окно выбора
- вывод данных в таблице
- построение графика по дням
- минимальная температура — синим
- максимальная температура — красным
- расчет дня с самым сильным и самым слабым перепадом
- прогноз на N дней методом скользящего среднего
- экспорт графика в PNG
- выбор периода по датам
- масштабирование графика выделением области мышью

## Запуск

1. Открыть проект в Visual Studio
2. Убедиться, что выбран тип проекта Windows Forms App (.NET Framework)
3. Собрать проект
4. Запустить `Program.cs`

## Формат CSV

Колонки:

- date
- min_temp
- max_temp
- avg_temp
- weather

Дата должна быть в формате `yyyy-MM-dd`.

# Анализ потребительских расходов

Windows Forms-приложение для анализа данных о потребительских расходах.

## Возможности

- загрузка CSV файла с данными о расходах
- вывод данных в таблице (год, расходы, процент изменения)
- построение линейного графика расходов по годам
- расчет среднегодового темпа роста
- определение года с максимальными и минимальными расходами
- прогноз на 3 года методом линейной экстраполяции
- экспорт графика в PNG
- отображение факта и прогноза на одном графике (синий — факт, красный пунктир — прогноз)

## Запуск

1. Открыть проект в Visual Studio
2. Убедиться, что выбран тип проекта **Windows Forms App (.NET Framework)**
3. Собрать проект (Build → Build Solution)
4. Запустить через `Program.cs` (F5)

## Формат CSV

Файл должен называться `expenses.csv` и находиться в папке `Data` проекта.

Колонки:

| Колонка | Описание |
|---------|----------|
| Год     | Год в формате YYYY |
| Расходы | Сумма расходов в рублях |
